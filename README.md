## Overview

This repository is the full replication package for the paper. We estimate heterogeneous conditional average treatment effects (CATEs) of high social protection coverage on annual changes in the Gini coefficient, using a **TabNet–CausalForestDML** (Double Machine Learning) estimator on an unbalanced panel of **25 Asian economies, 1990–2024**.

### Key findings

| Finding | Value |
|---|---|
| Significant inequality-reducing CATEs | 5 countries: Lao PDR (−0.388), Nepal (−0.379), Bangladesh (−0.336), Vietnam (−0.327), China (−0.314) Gini points/year |
| Dominant moderator (SHAP) | Rule of law |
| Robustness | All 13 specifications negative; 18/18 LOCO estimates negative; placebo p = 0.010 |
| Break-even coverage threshold | 8.3% of most vulnerable population (BCR > 1) |

## Data Sources

All data are publicly available. Raw files are **not** tracked by git; see `data/raw/README_data.md` for download instructions.

| Variable group | Source | URL |
|---|---|---|
| Gini coefficient; GDP per capita (PPP) | World Bank WDI | https://databank.worldbank.org/source/world-development-indicators |
| Social protection coverage | World Bank ASPIRE | https://www.worldbank.org/en/topic/socialprotection/brief/atlas-social-protection-resilience-equity |

## Replication Instructions

### 1. Environment setup

```bash
git clone https://github.com/canhtq-stack/social-protection-climate-asia.git
cd social-protection-climate-asia
pip install -r requirements.txt
```

### 2. Download raw data

Follow the instructions in `data/raw/README_data.md` for each of the 6 data sources. Place downloaded files in `data/raw/` as specified.

### 3. Build the panel dataset

```bash
python src/01_data_build.py
```

Output: `data/processed/panel_asia_1990_2024.csv` (~875 country-year observations)

### 4. Main estimation — Table 3 and Figure 1

```bash
python src/02_causal_forest_dml.py
```

⏱ Estimated runtime: ~45 minutes (Intel Core i7, 32 GB RAM)

### 5. SHAP analysis — Table 2 and Figure 2

```bash
python src/03_shap_analysis.py
```

### 6. Robustness checks — Table 5 and Appendix A–C

```bash
python src/04_robustness_checks.py
```

### 7. Break-even analysis — Section 6.1

```bash
python src/05_breakeven_analysis.py
```

### Interactive notebook

All steps are also available in:

```bash
jupyter notebook notebooks/main_analysis.ipynb
```
